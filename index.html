📖 Mode d'utilisation

À quoi sert cette page

Cette page est une application de prise de notes locale dans le navigateur. Elle permet de :

  • créer des notes
  • les classer par catégorie
  • ajouter des tags
  • choisir une couleur
  • choisir une image de fond
  • marquer des notes en favori
  • joindre un fichier
  • rechercher, filtrer et trier les notes
  • déplacer les notes dans la corbeille, puis les restaurer ou les supprimer définitivement

Les données sont stockées dans le navigateur via IndexedDB, donc elles restent sur le poste utilisé, tant que le navigateur ne vide pas son stockage.

1. Ouvrir l’application

Il suffit d’ouvrir le fichier HTML dans un navigateur moderne, par exemple : Chrome, Edge, Firefox.

2. Créer une nouvelle note

Clique sur ➕ Nouvelle note. Une fenêtre d’édition s’ouvre.

3. Ajouter une pièce jointe

Formats autorisés : PDF, DOC, DOCX, PNG, JPG, JPEG. Taille maximale : 12 Mo.

4. Ajouter des emojis

Le panneau emoji permet d’insérer des icônes dans le titre, les tags ou le contenu.

5. Choisir la couleur ou le fond

Tu peux utiliser une couleur prédéfinie, la palette libre, ou un fond image parmi les modèles proposés.

6. Favoris

Une note favorite peut être retrouvée rapidement via le filtre dédié.

7. Recherche, filtres et tri

Utilise la recherche, les catégories et le tri pour naviguer dans tes notes.

8. Corbeille

Les notes supprimées sont d’abord déplacées dans la corbeille, puis peuvent être restaurées ou supprimées définitivement.

9. Stockage

Les données sont enregistrées localement dans le navigateur.
